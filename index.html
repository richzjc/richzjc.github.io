kotlin 基础语法

发表于 2018-05-20 | 分类于 kotlin |

基础语法

1. 默认导入

kotlin.*
kotlin.annotation.*
kotlin.collections.*
kotlin.comparisons.*
kotlin.io.*
kotlin.ranges.*
kotlin.sequences.*
kotlin.text.*

2. 函数定义

fun sum(a: Int, b: Int): Int {   // Int 参数，返回值 Int
    return a + b
}

3. 关键字

Unit: 表示没有返回值 
vararg: 可变长参数
var: 可变变量
val: 不可变变量
is: 类型判断（可在前面加上感叹号）
in: 区间判断（同样可以在前面加上感叹号）

4. 字符串模板

$ 表示一个变量名或者变量值

$varName 表示变量值

${varName.fun()} 表示变量的方法返回值:

5. 空指针检查

//类型后面加?表示可为空
var age: String? = "23" 
//抛出空指针异常
val ages = age!!.toInt()
//不做处理返回 null
val ages1 = age?.toInt()
//age为空返回-1
val ages2 = age?.toInt() ?: -1

6. 区间判断

记住下面这几种写法即可

if (i in 1..10) { // 等同于 1 <= i && i <= 10
    println(i)
}

// 使用 step 指定步长
for (i in 1..4 step 2) print(i) // 输出“13”

for (i in 4 downTo 1 step 2) print(i) // 输出“42”

// 使用 until 函数排除结束元素
for (i in 1 until 10) {   // i in [1, 10) 排除了 10
     println(i)
}
